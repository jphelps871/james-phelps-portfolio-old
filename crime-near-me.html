Crime near me

The Project

The apps aim was to give users the ability to search if any crime occured at their specified location, the objectives were to: Allow users to search a specific location, return results in a simplistic manner, give options to save returned data into a database.

Agile devlopment

Creating a simple 2 day sprint helped form the main design idea, the focus would be simplicity, presenting the the users only the information they needed.

Prototype to development

The aim was not to make a pixel perfect design; instead allowing automatic responsive design, such as flex-box to auto size components.

Developed design

Bellow is an example of the home page, and main section for for the site. After a user has input data, a component is automatically generated from the api and displayed using ‘map’ and Reacts ‘props’ feature

Other Features

Login page, uses mongodb and node to both register and login a user. Users are given a JSON web token, stored in a cookie and sent via http to validate the user when they access user only content like ‘saved data’ page.

React

Using React for the project helped in the ability to easily store states which changed throughout the app.


Furthermore, having independent components was incremental in displaying crime data dynamically, which helped reduce code and increased speed.

Node.js

A backend server using node was set up as a proxy. Using a MVC approach to facilate the users response, create routes and return the relavent data.


Using a noSQL server (mongodb) helped in the overal MERN design of the project. The simplistic, single programming approach helped with documentation of the project.
